# Security Policy

## Reporting a Vulnerability

Payroc takes the security of our products and SDKs seriously.

If you believe you have discovered a security vulnerability in this repository or any Payroc SDK, **please do not open a public GitHub issue**.

Instead, report the issue in accordance with our official Security Disclosure Policy:
